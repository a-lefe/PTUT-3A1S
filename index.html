Auteurs

Ce site a été réalisé par Jerome Alibert, Arthur Vigne, Alexandre Lefebvre, Baptise Guégan et Sébastien Rigaud.

Description

Il permet de visualiser simplement tous les vols au départ de l'Aéroport Saint-Exupéry de Lyon. Vous pouvez également, dans l'onglet Statistiques, consulter différentes statistiques sur les vols, et visualiser ces données sous forme graphique

Données

Le site a été réalisé à l'aide du flux de données fournis par Data Grand Lyon que nous exploitons en continu afin de fournir des informations en temps réel.

Technologies

Le site à entièrement été réalise en PHP, JavaScript, HTML 5, Chart.js pour la gestion des graphiques et le framework CSS Material Design Lite.
